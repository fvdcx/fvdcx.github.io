leetcode-buy-stock

发表于 2016-07-31   |   分类于 算法 ， leetcode   |  

题目大意

　　https://leetcode.com/problems/best-time-to-buy-and-sell-stock//

　　你有n天股票的价格，现在可以选择在某天买入，某天卖出，求出能获得的最大利润

题目分析

　　数组s[i]代表可选天数是1~i天时，能获得的股票最大利润

　　s[i]=max{ max{ (v[i] - min{v[0]....v[i-1]}), 0 }, s[i-1] }

　　满足动态规划的条件

代码

　　时间复杂度O(n)，额外空间复杂度O(1)

计算数字的二进制中1的个数
