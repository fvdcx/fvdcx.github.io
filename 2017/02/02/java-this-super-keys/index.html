super关键字

　　Java中super关键字通常有以下几种用途：

  1. 调用父类的方法

  2. 在构造器中调用父类构造器方法

    需要注意的是，如果在构造器中利用super调用超类的构造器，super语句要位于方法的第一条。而且如果子类的构造器没有显示地调用超类构造器，会自动调用超类的默认构造器。如果超类没有默认构造器，子类构造器中又没有显示调用超类的构造器，那么编译器会报错。
